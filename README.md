# JmalCoder APK Project

Your Kotlin code is inside `app/src/main/java/com/kotlinstudio/app/MainActivity.kt`

## ⭐ Easiest: GitHub Actions (free, no PC needed)
1. Create a free account at github.com
2. Create a new repository
3. Upload all these files to the repo
4. Go to the **Actions** tab
5. Wait ~3 minutes for the build to finish
6. Click the completed run → scroll to **Artifacts** → download **apk-debug**
7. Install the APK on your Android device!

## Android Studio
Unzip → Open folder → Build → Build APK(s)

## Command Line
```
chmod +x ./gradlew
./gradlew assembleDebug
```
APK: `app/build/outputs/apk/debug/app-debug.apk`
